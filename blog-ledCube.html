Project

8X8X8 Led cube

Dic 2020

Description

This is a led cube that was implemented using VHDL and a DE-2 board, the purpose of this project was to use the basic of digital systems and implement all the knowledge acquired about this topic. This led cube was made to follow the beat of a song, Into the void - kiss, and making different patterns by turning on and off its leds. All the project was made form scratch including the cube assembling. The leds used on this project were of 20mA and 3.2V

Used Technologies

  • VHDL
  • DE-2 Altera board

  • YouTube video

What I learned

In this project I learned how to use multiplexers in order to select the output that the cube was going to have depending on a global clock. I also know how to make frequency dividers by implementing counters in VHDL and I know how to properly use the board DE-2. Finally learned to solve problems under pressure and to find different ways to get the desired result and choose which one was the best for being implemented.
